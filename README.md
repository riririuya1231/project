# 雅思产品匹配工具 / 雅思课程咨询方案生成器

一个无需构建的静态网页内部顾问工具。页面保留 main 分支已有的雅思分数换算能力，可先把托福、四六级、紫藤雅思入班测试、考研英语、高考英语换算成参考雅思分，再根据学生当前雅思总分、目标总分、小分短板、线上/线下偏好、班型偏好和预算敏感度，自动匹配高顿紫藤雅思产品并生成可复制咨询方案。

## 功能

- 雅思分数换算：支持老托福/TOEFL iBT、新托福、英语四级、英语六级、紫藤雅思入班 60 分钟测试、考研英语和高考英语。
- 高考英语支持按 150/120/100 分制自动折算到 150 分制后匹配参考雅思分。
- 换算结果可一键填入“当前雅思总分”，继续生成 IELTS 产品匹配方案。
- 输入当前雅思总分与目标雅思总分，自动计算总分差距。
- 小分选填：可填写听力、阅读、写作、口语；未填写时提醒补充小分或先做入学测试。
- 根据小分短板判断听力/阅读输入项、写作/口语输出项、总分够但小分不够、四科都低等情况。
- 每次至少输出主推方案、性价比方案、高保障方案。
- 产品库区分线上大班、线上小班、线下班课、线上/线下 1V1、班课+1V1、飞跃 A/B/C 计划、单项课，并在产品速览区单独展示线上大班课与1V1价格阶梯。
- 推荐 1V1 时按“6–18H / 20–39H / 40–59H / 60H+”阶梯单价自动计算推荐小时数、最低单价、最低总价、官方参考总价、节省金额和 CP 码。
- 推荐班课+1V1 时写清先上什么班课、班课解决什么、后续多少小时 1V1、重点补什么和组合价格；选择某一班型偏好时，仅输出该班型方向的方案，避免选1V1却继续推荐班课。
- 生成“内部顾问版方案”和“发给学生/家长版话术”，支持一键复制。
- 产品表支持按线上/线下、班型、当前分、目标分、预算筛选，并可展开查看课程详情；推荐结果和产品表都会写明总课时、主课、直播/面授、录播、学辅期、服务和赠送资料。

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 产品数据维护

课程数据集中维护在 `script.js` 的 `products` 数组和 `serviceBundles` 对象中。后续如果官方价、最低价、课时、CP 码、服务内容、赠送资料或重读/补偿规则有更新，可直接替换对应字段。

分数换算数据集中维护在 `script.js` 的 `conversionConfigs` 对象中。托福/四六级、内部 60 分钟测试按图中区间录入；考研英语和高考英语为咨询内部分层参考，可按后续校区标准调整区间。
