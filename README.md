# 高顿紫藤雅思产品匹配工具

顾问内部自用的静态页面工具。输入学生当前雅思总分、目标总分、小分、线上/线下偏好、班型偏好、预算敏感度、备考用途和是否接受测评/试听后，系统会自动匹配高顿紫藤雅思产品，并生成可复制的“内部顾问版方案”和“发给学生/家长版话术”。

## 功能

- 支持当前总分、目标总分、听力/阅读/写作/口语小分选填。
- 支持线上 / 线下 / 都可以、大班 / 小班 / 1V1 / 班课+1V1 / 飞跃计划 / 都可以、预算优先 / 效果优先 / 高保障优先。
- 产品库按线上大班、线上小班、线下班课、1V1、班课+1V1、飞跃A/B/C计划、单项课拆分。
- 每次至少输出主推方案、性价比方案、高保障方案。
- 每个方案展示推荐课程名称、班型、上课形式、适合原因、学习顺序、课时拆分、价格、CP码、优势、服务、赠品、重读/补偿、风险提示和下一步建议。
- 1V1 严格按阶梯价自动计算：`<20小时 720元/H`、`20–39小时 680元/H`、`40–59小时 650元/H`、`≥60小时 600元/H`，并展示官方参考总价、最低总价、节省金额和 CP 码。
- 班课+1V1 会写清先上哪个班课、班课解决什么问题、阶段目标、后续多少小时 1V1、1V1 重点补什么和组合总价。
- 下方产品库支持按线上/线下、班型、当前分、目标分、预算筛选并查看课程详情。

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 产品数据维护

产品数据集中维护在 `script.js` 的 `products` 数组中；1V1 阶梯价维护在 `hourlyPriceTiers`，CP 码维护在 `oneOnOneCpMap`。如产品价格、课时、CP码、服务、赠品或重读/补偿规则更新，直接替换对应字段即可。
